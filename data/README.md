# GeoDots Data

- configdirs = purely .config directories GeoDots uses
- dirs = config, and other directories that GeoDots uses

#

- errormsg = message for when a version is affected by a breaking change
- errorver = targeted 'broken' version due to a breaking change

#

- pkg-pacman = list of packages to install
- pkg-aurs = list of AUR packages to install
- pkg-gtk = list of GTK packages to install (if selected)
- pkg-qt = list of QT packages to install (if selected)
- pkg-conflicts = list of conflicting packages, if they exist

#

- version = current version of GeoDots
